# `GET /api/v1/books/price`

### Description:
This endpoint allows users to retrieve a list of books within a specified price range and other parameters.

### Parameters:
- `min_price` (optional): Minimum price of the books to be retrieved. Default is `0`.
- `max_price` (optional): Maximum price of the books to be retrieved. Default is `no max price`.
- `page` (optional): Page number for pagination. Default is `1`.
- `limit` (optional): Number of books per page. Default is `10`.
- `sort_order` (optional): Sorting order of the results. `1` for ascending and `-1` for descending. Default is `1`.

### Response

The response will show first the lower prices (ascending order is 1 and it was the order provided). Also will only show the books within the price range of 10 and 50.
The results will be only for the page 1 and the limit will be 10 results. (because, they're only 2 results to show, that won't make a difference)
