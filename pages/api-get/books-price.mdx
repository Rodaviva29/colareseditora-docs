# `GET /api/v1/books/price`

### Description:
This endpoint allows users to retrieve a list of books within a specified price range.

### Parameters:
- `min_price` (optional): Minimum price of the books to be retrieved. Default is `0`.
- `max_price` (optional): Maximum price of the books to be retrieved. Default is `no max price`.
- `page` (optional): Page number for pagination. Default is `1`.
- `limit` (optional): Number of books per page. Default is `10`.
- `sort_order` (optional): Sorting order of the results. `1` for ascending and `-1` for descending. Default is `1`.

### Examples

#### Right Request

```http copy
GET /api/v1/books/price?min_price=10&max_price=50&page=2&limit=20&sort_order=-1
```

#### Wrong Response (Invalid Sort Order Value):

```http
GET /api/v1/books/price?min_price=ten&max_price=50&page=2&limit=20&sort_order=-1
```

#### Follow up Response

```json
{
    "error": "Invalid parameters provided."
}
```
