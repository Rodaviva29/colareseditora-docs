# `GET /api/v1/books/<string:title>`

### Description:
This endpoint allows users to retrieve a list of books based on a specified title.

### Parameters:

All parameters are optional, if there is no specific value provided, default will be used.

- `page`: Page number for pagination. Default is `1`.
- `limit`: Number of books per page. Default is `10`.
- `sort_order`: Sorting order of the results. `1` for ascending and `-1` for descending. Default is `1`.

### Response

The response will show books that match the specified title, with pagination details. For example, if the title is "Example Book" there are 2 matching results.
